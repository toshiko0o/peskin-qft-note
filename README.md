# Peskin-QFT-note
Peskin &amp; Schroeder QFT 中文笔记

# 前言

　　本文为基于 *An Introduction to Quantum Field Theory - Peskin & Schroeder* 一书的量子场论笔记。 

　　本文主要对此书正文部分一些复杂的计算和难以理解的段落作了注释，同时根据研究生课程内容对部分较为简略的段落添加了补充说明。由于说明较为困难，故此笔记将以中文呈现。若后有闲暇，可能翻译为英文。

　　如发现任何错误或遗漏，或希望与我讨论书中的内容，欢迎通过[邮件](mailto:toshiko.h19@gmail.com)与我联系。
